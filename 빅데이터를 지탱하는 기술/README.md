# 책의 주제

> 자동회된 데이터 처리 (!= 데이터 분석)

- `데이터의 활용 방법`이 아닌 `데이터 처리를 어떻게 시스템화` 하는가
- 대상은 `빅데이터의 기술을 배우고자 하는 엔지니어`, `데이터 과학자` (깊이 있게 다루지는 않음)
- 목적은 기본적인 데이터 처리의 개념을 폭넓게
- 데이터 분야를 처음 공부하는 사람들에게 유용한 책
- 파이썬과 SQL에 대한 기본 지식 필요
